# 조건 연산자 ?

조건(?:) 연산자는 주어진 조건의 참과 거짓에 따라 값을 선택한다

- 조건부 연산자는 물음표 `?` 로 표시하며 피연산자가 세 개 이기 때문에 **‘삼항 연산자’**라고도 부른다.
- 자바스크립트에서 피연산자가 3개나 받는 연산자는 조건부 연산자가 유일하다.

예시1)

```java

let adult = "성인입니다"
let child = "청소년입니다"
let result = (age > 18) ? adult : child;
```

예시2(피연산자)

```java
let baby = "어린아이입니다"
let adult = "성인입니다"
let child = "청소년입니다"
let result = (age < 10) ? baby :
	(age <18) ? child :
	adult;
```
